SideM風会話ジェネレータ

アイドルマスター SideM にある、マイデスクでアイドル同士が会話するシステム。あれが シンデレラガールズ や ミリオンライブ！ にもあったらなぁ…。ということで作ってみた。

遊び方

  1. 各コマに画像ファイルをドラッグアンドドロップで入れます。
    画像ファイルは、ブラウザーで表示中のファイルやパソコンに保存されているファイルが使えます。
  2. 各コマをドラッグして画像の位置を調整します。
  3. 各コマの上でマウスホイールを動かして画像を拡大/縮小します。
  4. 名前と台詞を入力します。
  5. 完成した画像を右クリックして保存します。
  6. 画像とハッシュタグ「#SideM風会話」を付けてツイートすると作者が喜びます。 Tweet

動作環境

  • Internet Explorer 11、Mozilla Firefox 27、Google Chrome 33で動作する事を確認しています。
  • Internet Explorer 9で動作しない事を確認しています。
canvasに対応したブラウザーでご覧下さい。
